MARVELMADE fundraising web app

summary

Marvelmade is a project I was assigned during my Bloc apprenticeship. The project required us to create a fundraising web app in which two primary user groups, fundraisers and donors, could collaborate to crowdfund charities and fundraisers for specific causes.

I opted to create a fundraising platform that focused on education-based causes, and allowed for both monetary and time donations. Through creating and conducting user surveys, personas, and flows, I was able to then wireframe, design, and develop the identity and front-end framework for the Marvelmade web app.

problem

While a number of fundraising platforms already exist, Bloc posed us the challenge of creating an improved, original version of what had already been done.

user research

The first step in executing a solution was to conduct user research. To launch this research phase, I created user surveys for both potential donors and fundraisers. I analyzed survey results to determine what features these potential users thought necessary. I established that prospective users were not only interested in donating money, but also time, especially in the realm of something as hands-on as education. From there, I was able to illustrate a series of representative personas, as well as user flow diagrams to establish the donor and campaign-creator processes.
